# findDoc 執行流程

```
/findDoc 執行流程
│
├─ 1. 【輸入判斷】
│   ├─ 以 "https://www.notion.so/" 開頭 → URL 查詢模式
│   └─ 其他 → 關鍵字查詢模式
│
├─ 2A. 【URL 查詢模式】
│   ├─ 2A.1 從 URL 提取 page_id（32 字元 hex）
│   ├─ 2A.2 用 rg 搜尋 page_id
│   │   ├─ 找到 → 繼續
│   │   └─ 沒找到 → 提示用 /exportN 匯出
│   ├─ 2A.3 關聯文件發現（用日期前綴 MMDD_N）
│   └─ 2A.4 批次讀取並摘要輸出
│
├─ 2B. 【關鍵字查詢模式】
│   ├─ 2B.0 前置處理：跳脫 [] 特殊字元
│   ├─ 2B.1 第一階段：搜尋 *_api/ 目錄
│   │   ├─ 有結果 → 讀取並摘要
│   │   └─ 無結果 → 進入第二階段
│   └─ 2B.2 第二階段：搜尋整個 prompts 專案
│
└─ 3. 【輸出結果】
    ├─ 找到 → 文件清單 + 內容摘要
    └─ 沒找到 → 建議其他搜尋方式
```

## 特殊字元處理

| 原始關鍵字 | 跳脫後 |
|-----------|--------|
| `[物件資料管理]` | `\[物件資料管理\]` |
| `[權限]` | `\[權限\]` |
| `customer` | `customer`（無需跳脫）|
